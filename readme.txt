== Description ==

You love to create content in Wordpress. But your articles only become really good with a suitable and special image. And this is where Zoonar Image Search comes to the rescue. Search our huge stock for the right image. 
Buy directly in Wordpress and pay comfortably via PayPal. The images are transferred to your media library with a click. Do you have several Wordpress websites? Then use your already purchased images there as well. It's as simple as eating pancakes.

[For more infos, check our Site](https://www.zoonar.com)

== Installation ==

You need:
* WordPress Version: 4.7 or higher
* a valid PayPal account for purchasing images

How to:
* Log into WordPress
* Under Plugins search for "Zoonar"
* Install the Plugin

Important: The Plugin requires the JSON PHP extension
